# 🎬 Movie Watchlist

Веб-застосунок для відстеження фільмів — зберігай списки "до перегляду" та "переглянуто".

## Можливості

- ➕ Додавання фільмів з назвою, жанром, роком та постером
- 🖼️ Постер — посилання або завантаження з комп'ютера
- 🎭 Вибір жанру зі списку або власний жанр
- ✅ Зміна статусу фільму (до перегляду / переглянуто)
- 🗑️ Видалення фільму
- 🔍 Фільтрація за статусом
- 💾 Збереження даних у localStorage

## Технології

- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Nuxt UI 4](https://ui.nuxt.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Pinia](https://pinia.vuejs.org/)

## Запуск

1. Клонуй репозиторій:
```bash
git clone https://github.com/k-skrypko/movie-watchlist.git
cd movie-watchlist
```

2. Встанови залежності:
```bash
npm install
```

3. Запусти застосунок:
```bash
npm run dev
```

4. Відкрий у браузері: `http://localhost:5173`

## Скрипти

| Команда | Опис |
|---|---|
| `npm run dev` | Запуск в режимі розробки |
| `npm run build` | Збірка для продакшену |
| `npm run preview` | Перегляд збірки |
